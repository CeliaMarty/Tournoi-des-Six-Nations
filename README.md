# Rugby BI — Six Nations (2000-2024)  

Ce projet analyse l’évolution offensive et le déséquilibre des matchs du Tournoi des Six Nations.  

Le dossier contient :  
Préparation et exploration du dataset rugby_six_nations.csv  
Création de nouvelles métriques : points totaux, différence de score, matchs serrés, bonus offensifs  
Analyse graphique de l’évolution des points moyens, des matchs à score élevé et du nombre de bonus  
Étude de l’équilibre des matchs et des performances des équipes par décennie  
Dashboard interactif Streamlit pour visualiser les KPIs, tendances et heatmaps par équipe et par année  

Le code est prêt à être exécuté localement ou dans un environnement Streamlit.
